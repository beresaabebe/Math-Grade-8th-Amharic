# Maths Grade 8 Amharic

A professional Android application for Grade 8 Mathematics in Amharic.

## Features
- **Modern UI/UX**: Professional design with ViewPager2 for chapter navigation.
- **Native PDF Renderer**: High-performance PDF viewing with zoom support.
- **Ads Integration**: AdMob with Facebook and Unity mediation for maximum revenue.
- **In-App Updates**: Stay up to date with the latest features and fixes.
- **In-App Review**: Rate the app easily within the application.
- **16KB Page Support**: Fully compatible with modern Android 15+ devices.
